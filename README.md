#  Secure Full-Stack Authentication System

##  Description

A full-stack web application implementing secure authentication using JWT and bcrypt, with protection against common vulnerabilities.

##  Features

* User Signup & Login
* Password hashing (bcrypt)
* JWT authentication
* Protected routes
* Secure API handling

##  Security Concepts Used

* Password hashing
* Token-based authentication
* Protection against unauthorized access

##  Tech Stack

* React
* Node.js
* Express
* MongoDB
